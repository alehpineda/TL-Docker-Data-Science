# Comandos utilizados

## Iniciar jupyter notebooks
```bash
docker container run -d -p 8888:8888 --name tl-jupyter jupyter/scipy-notebook
```

## Iniciar superset
```bash
docker container run -d -p 8088:8088 --name tl-superset tylerfowler/superset
```

## Iniciar docker compose
```bash
docker container run -d -p 8888:8888 --name tl-jupyter jupyter/scipy-notebook
```
